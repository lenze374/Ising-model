#### 格点数量的影响
在进行模拟时，发现需要达到100x100的格点数量才能看到比较明显的相变现象，此时模拟的结果会开始变得尖锐，与理论预期较为吻合。而在较小的格点数量下（如20x20），相变现象并不明显，结果较为平滑。这个现象相当有趣，反映了尺寸对系统行为的影响。
另一方面，最开始采取了热启动的方式，即使用随机的格点网络开始演化。这个情况下可能会形成磁畴，从而使低温下的模拟结果有明显的异常点。为了避免这个问题，改为冷启动，即所有格点初始时刻都取相同的自旋方向（+1或-1）。这种方式下，低温下的模拟结果变得更加平滑，符合预期的物理行为。当然，更物理的方法使使用退火算法，即降低温度后使用前一温度下的最终状态作为下一温度的初始状态，这样可以更好地模拟实际物理过程。
